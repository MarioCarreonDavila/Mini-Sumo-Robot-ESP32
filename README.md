# Mini-Sumo-Robot-ESP32
Mini sumo robot using ESP32 for competition in 2024

This project consists of a mini sumo robot that is suposed to be capable of search and detect the enemy 
robot, so it can take it out of the dojo. The dojo is the circular area where the robots "fight" trying
to take the other one out.

# Features 
2 DC Pololu motors
